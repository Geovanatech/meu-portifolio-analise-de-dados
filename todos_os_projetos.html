Dashboard de Vendas

✅ Finalizado RFM & Análise

De dados brutos a insights estratégicos

Análise de clientes com RFM e tratamento de outliers via IQR para identificar padrões de consumo. Identifiquei que 13% dos clientes geram ~30% da receita, revelando oportunidades estratégicas para segmentação e marketing.

💡 Aprendizado Principal: A limpeza de dados é uma etapa estratégica da análise, onde padrões relevantes emergem. No contexto de marketing, valores atípicos podem representar os segmentos de maior valor para o negócio.

Ferramentas: Excel, Power Query (M), Power BI (DAX) e IA
